# Lumière - Fine Dining Restaurant Website

A premium restaurant website built with Next.js, React, and Tailwind CSS, featuring:

- **Hero Section**: Full-bleed cinematic fine-dining interior image with prominent reservations CTA
- **Menu Showcase**: Three-column card layout for Appetizers, Main Courses, and Desserts
- **About Section**: Restaurant narrative emphasizing craftsmanship and hospitality
- **Reservations**: Functional booking form with date/time/guest selection
- **Footer**: Contact information, hours, and social links
- **Responsive Design**: Mobile-first approach with semantic HTML and accessibility standards

## Features

✦ Cinematic hero image from fal-ai/flux/dev (provided URL)
✦ Mobile-responsive design with Tailwind CSS
✦ Warm color palette (golds, deep creams, soft shadows)
✦ OG meta tags for social sharing
✦ Accessibility-first: semantic HTML, alt text, contrast standards
✦ Performance optimized: Next.js Image component with lazy loading
✦ Lighthouse-ready: modern CSS, minimal JavaScript, fast image delivery

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS 3
- **UI**: React 18
- **Image Hosting**: fal-ai media CDN

## Performance Targets

- Lighthouse Score: >80 across all categories
